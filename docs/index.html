Critical features of email clients

Reading panes

The reading pane should focus on the latest selected message, especially during multiselection. The best email client that handles this use case correctly is Windows Live Mail 2012 16.4.3528.0331, followed by eM Client 7.1.32088.0, although eM Client's reading pane transition was not as smooth. Outlook 2016 16.0.4266.1003, Outlook Mail, Thunderbird 52.7.0, and Mailspring 1.2.1-7e7447b6 do not follow this use case correctly.

Scrollbars

The right vertical scrollbar must obey Fitts' law, especially when the window is maximized. Moving the scrollbar to the edge of the screen creates a target area of "infinite width".

When the window is maximized, the preview pane scollbar should be placed at the edge of the screen. This allows the user to click the scrollbar effortlessly because no matter how far the mouse pointer moves to the edge of the screen, it will always land on the scrollbar. The scrollbar will then have "infinite width".

All major web browsers obey Fitts' law for the window scrollbar.
